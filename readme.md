# 프로젝트
## 프로젝트 개요
"UPBIT" 라는 가상 기업을 주제로 jsp의 디렉티브, 스크립트, 표현식만 확용하여 기업형
데스크탑 웹 애플리케이션을 작성하였으며, 해당 프로젝트는 회원기능인 회원가입, 로그인, 로그아웃, 회원정보확인을 구현해두었음
